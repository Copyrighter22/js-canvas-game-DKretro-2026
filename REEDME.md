# Donkey Kong Retro

## Descripción
Este proyecto es una recreación del clásico videojuego Donkey Kong (la versión donde llevas a Mario y esquivas barriles) desarrollado en JavaScript. El objetivo principal es aprender conceptos de programación, incluyendo la estructura del código, la programación orientada a objetos (POO) y el uso del elemento Canvas de HTML5 para renderizar gráficos.

## Características
- **Jugabilidad Clásica**: Controla a Mario para subir plataformas, evitar barriles y salvar a la princesa.
- **Programación Orientada a Objetos**: Implementación de clases para personajes, objetos y lógica del juego.
- **Canvas Rendering**: Uso de Canvas para dibujar sprites, animaciones y colisiones en tiempo real.
- **Aprendizaje**: Proyecto educativo para practicar JavaScript, HTML y CSS.

## Tecnologías Utilizadas
- **JavaScript**: Lógica del juego y POO.
- **HTML5 Canvas**: Para gráficos y animaciones.
- **CSS**: Estilos básicos.
- **Herramientas**: Editor de código (VS Code recomendado), navegador web moderno.

## Cómo Ejecutar
1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en un navegador web.
3. Usa las teclas de flecha para mover a Mario y espacio para saltar.

## Estructura del Proyecto
- `index.html`: Archivo principal HTML.
- `style.css`: Estilos CSS.
- `script.js`: Código JavaScript con clases y lógica del juego.
- `assets/`: Carpeta para imágenes y sonidos (si se incluyen).

## Aprendizajes Clave
- Creación de clases en JavaScript (e.g., `Player`, `Barrel`, `Platform`).
- Manejo de eventos del teclado.
- Detección de colisiones.
- Animaciones.
- Estructura modular del código.
